# Release Notes v0.2.0-master

日期：2026-07-02

## 摘要

本版本為目前 Master 專案紀錄版本，重點是完成 SharePoint 後台清單、Power Automate 功能測試、Teams Adaptive Card 設計，以及通知時間規則補強。

## 完成內容

- 在 `ALP_TW_AD` 建立 SharePoint List：`公務車借用管理`。
- 完成主要欄位建立與驗證。
- 新增 `是否整天` 欄位。
- 新增 `預計通知時間` 欄位。
- 建立 Power Automate 功能測試流程。
- 完成 SharePoint 與 Office 365 Outlook 標準連接器基礎測試。
- 完成 Teams Adaptive Card JSON 範例。
- 更新通知規則為借用前 1 小時，但最早 08:00；整天借用 08:00。
- 建立 Master 專案紀錄。
