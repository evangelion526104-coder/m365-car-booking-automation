# Release Notes v0.2.1-safeguards

日期：2026-07-02

## 摘要

本版本補強正式上線前必須完成的系統防呆設計，避免時區錯誤、資料重複、舊卡片覆蓋資料，以及 Outlook 取消或異動未同步 SharePoint 的風險。

## 完成內容

- 新增系統防呆設計文件：`docs/system-safeguards.md`。
- 明確列出五項正式上線前必須完成的防呆機制：
  - 統一時區為 `Asia/Taipei`。
  - 以 Outlook Event ID / iCalUId 作為唯一識別碼。
  - Outlook 取消與異動同步 SharePoint。
  - 舊 Teams Adaptive Card 回覆前失效檢查。
  - Flow 執行鎖與重複資料防止。
- 補強 SharePoint List 欄位設計。
- 補強 Power Automate 同步、通知、回覆前檢查與重複防止流程。
- 補強 Teams Adaptive Card 回覆前檢查邏輯。
- 新增防呆測試案例 SG-T01 至 SG-T10。
- 更新 Master 紀錄、里程碑、完成清單與待辦事項。

## 尚未完成

- 本版本建立時尚未實際在 SharePoint 線上清單新增所有防呆欄位；此項已於後續 `v0.2.3` 完成。
- 尚未完成正式 Power Automate 流程實作。
- 尚未完成 SG-T01 至 SG-T10 實測。
- 尚未完成正式上線驗收。
